## Get started in 4 steps

Follow these steps to go from zero to your first successful API response.

<Steps>
  <Step title="Create an account">
    Sign up at [dashboard.google.com](https://dashboard.google.com) to get access to your developer workspace. Your account gives you a personal dashboard where you can manage API keys, monitor usage, and explore logs.
  </Step>
  <Step title="Generate an API key">
    Inside the dashboard, navigate to **Settings → API Keys** and click **New API Key**. Copy the key and store it somewhere secure — you'll use it to authenticate every request you make.
  </Step>
  <Step title="Make your first request">
    Use your preferred HTTP client or the code examples in the [Quickstart](/quickstart) to send your first authenticated request to `https://api.google.com/v1`. A successful response confirms your key is active and your integration is working.
  </Step>
  <Step title="Explore the API reference">
    Head to the [API Reference](/api-reference/introduction) to browse every available endpoint, inspect request parameters, and test calls directly from the docs.
  </Step>
</Steps>
