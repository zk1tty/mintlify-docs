By the end of this guide you'll have a working API key, a successful response from the Google API, and a clear picture of how to build from here. You don't need any special tooling — just a terminal and five minutes. Let's get you making real requests against a live API.

<Steps>
  <Step title="Create your account">
    Go to [dashboard.google.com](https://dashboard.google.com) and sign up for a free developer account. Once you're logged in, you'll land on your personal dashboard where you can manage keys, view request logs, and monitor usage. Navigate to **Settings → API Keys** to get ready for the next step.
  </Step>
  <Step title="Generate an API key">
    On the API Keys page, click **New API Key**. Give it a descriptive name — something like `development` or `my-first-integration` — then click **Create**. Copy the key shown on screen and store it somewhere safe, such as a password manager or a local `.env` file.

    <Warning>
      Your API key is a secret credential. Never commit it to source control, embed it in client-side code, or share it publicly. Anyone who holds your key can make authenticated requests on your behalf. If a key is ever exposed, revoke it immediately from the dashboard and generate a new one.
    </Warning>
  </Step>
  <Step title="Make your first request">
    With your key in hand, send a GET request to the `/resources` endpoint. Replace `YOUR_API_KEY` with the key you just copied.

    ```bash
    curl https://api.google.com/v1/resources \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```

    A `200 OK` response means your key is valid and your request reached the API successfully.
  </Step>
  <Step title="Inspect the response">
    The API returns a JSON object with a `data` array containing your resources and a `pagination` object to help you page through larger result sets.

    ```json
    {
      "data": [
        {
          "id": "res_01J...",
          "name": "My First Resource",
          "status": "active",
          "created_at": "2024-01-15T10:30:00Z"
        }
      ],
      "pagination": {
        "page": 1,
        "per_page": 20,
        "total": 1
      }
    }
    ```

    Every list endpoint in the API follows this same envelope shape, so the parsing logic you write here works everywhere.
  </Step>
  <Step title="Explore further">
    You've made your first authenticated request — here's where to go next.

    - **[API Reference](/api-reference/introduction)** — Browse every endpoint, inspect accepted parameters, and send test requests directly from the docs.
    - **[Authentication](/authentication)** — Learn how to rotate keys, handle authentication errors, and keep your credentials secure.
    - **[Error Handling](/guides/error-handling)** — Understand the error envelope and write resilient retry logic.
    - **[Webhooks](/guides/webhooks)** — Subscribe to events and receive real-time push notifications instead of polling.
  </Step>
</Steps>

<Tip>
  Store your API key in an environment variable rather than hardcoding it in your source files. Use `export GOOGLE_API_KEY="your_key_here"` in your shell profile, then reference it as `$GOOGLE_API_KEY` in scripts and `process.env.GOOGLE_API_KEY` in Node.js. This keeps secrets out of your codebase and makes rotating keys a one-line change.
</Tip>

## The same request in multiple languages

The curl example above maps directly to any HTTP library. Here's how to make the identical call in Python and JavaScript.

<CodeGroup>
  ```bash curl
  curl https://api.google.com/v1/resources \
    -H "Authorization: Bearer $GOOGLE_API_KEY"
  ```

  ```python Python
  import os
  import requests

  api_key = os.environ["GOOGLE_API_KEY"]

  response = requests.get(
      "https://api.google.com/v1/resources",
      headers={"Authorization": f"Bearer {api_key}"},
  )

  data = response.json()
  print(data)
  ```

  ```javascript JavaScript
  const apiKey = process.env.GOOGLE_API_KEY;

  const response = await fetch("https://api.google.com/v1/resources", {
    headers: {
      Authorization: `Bearer ${apiKey}`,
    },
  });

  const data = await response.json();
  console.log(data);
  ```
</CodeGroup>
